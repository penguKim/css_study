html5 추가 폼 요소들

1. input type="time" - 시간(오전/오후 시 분)을 지정하는 폼요소

2. input type="date" - 시간(날짜 년 월 일)을 지정하는 폼요소

min="n"과 max="n"로 최소/최대값 지정 가능


3. input type="datetime-local" - 시간+날짜를 지정하는 폼요소

4. input type="week" - 시간(년 주)을 지정하는 폼요소

5. input type="month" - 시간(년 월)을 지정하는 폼요소

6. input type="tel" - 연락처(전화번호)를 지정하는 폼요소

모바일브라우저에서 지원하는 경우 가상키패드가 숫자형식으로 바뀌고 전화걸기 가능

7. input type="email" - 연락처(이메일)를 지정하는 폼요소

브라우저가 알맞은 형식(@유무, @이후 완성)인지 확인해 줌
모바일브라우저에서 지원하는 경우 가상키패드에 @, .com 등이 나타남

8. input type="url" - 연락처(웹주소)를 지정하는 폼요소

브라우저가 알맞은 형식(http:// or https:// 유무)인지 확인해 줌
모바일브라우저에서 지원하는 경우 가상키패드에 .com 등이 나타남

9. input type="color" - 색상값을 지정하는 폼요소

10. input type="number" - 숫자값을 지정하는 폼요소

증감버튼이 생기고
min="n"과 max="n"로 최소/최대값을, step="n" 간격을 지정 가능

명
명(min="0")
명(max="20")
명(step="10")

11. input type="range" - 범위내의 값을 지정하는 폼요소




html5 추가된 속성들

1. autofocus(autofocus="autofocus") - 해당 요소가 자동 포커싱 됨

2. autocomplete="on/off" - 자동 완성기능을 켜거나 끔

3. required(required="required") - 필수입력요소로 지정

4. placeholder="" - 입력예시를 보여주거나 설명 등을 추가


5. readonly(readonly="readonly") - 읽기만 가능한 요소로 지정(값이 전송됨)

6. disabled(disabled="disabled") - 사용불가한 요소로 지정(값이 전송안됨)
