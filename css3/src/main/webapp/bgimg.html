background - 배경

  1. background-color:color값 ; 배경색상
  2. background-image:url("이미지경로"); 배경 이미지

background-image 2개 지정

background-image:url("이미지-1경로"), url("이미지-2경로");
(기타 세부속성들도 따로 지정하려면 ,를 추가 background-repeat:repeat, no-repeat ; )

background-image의 속성들

  • background-repeat: ; 반복
    1. repeat : 기본값
    2. repeat-x : 가로 방향으로 반복
    3. repeat-y : 세로 방향으로 반복
    4. no-repeat : 반복 안함
  • background-size: ; 크기
    1. cover : 지정한 요소를 다 덮도록 배경이미지를 확대/축소
    2. contain : 지정한 요소 안에 배경 이미지가 다 들어오도록 이미지를 확대/축소
    3. Xn Yn : 너비 값과 높이 값을 지정
  • background-position: ; 위치
    1. left/center/right top/center/bottom : 수평 수직 지정
    2. Xn Yn : 너비 값과 높이 값을 지정
  • background-attachment: ; 스크롤 여부
    1. scroll : 해당 영역의 이미지 고정
    2. fixed : 전체 영역에서 이미지 고정
    3. local : 내용을 스크롤하면 배경 이미지도 스크롤

background-attachment: ; 스크롤 여부

[ 특정영역에 스크롤바 생성시 ]
scroll (기본) 자신 영역에는 고정, 브라우저에는 스크롤.

local 자신 영역과 브라우저 모두 스크롤.
