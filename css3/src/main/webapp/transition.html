transition 단순한 애니메이션 효과

css
  • transition-property: ; 적용할 속성
  • transition-duration: ; (필수) 진행 시간
  • transition-delay: ; 지연 시간
  • transition-timing-function: ; 가속 감속 등
    1. ease 기본값. 천천히 시작하고 느리게 끝남.
    2. ease-in 시작을 느리게
    3. ease-out 끝을 느리게
    4. ease-in-out 시작과 끝을 느리게
    5. linear 등속
    6. cubic-bezier(n,n,n,n) 베지에곡선값을 이용
      cubic-bezier.com 생성기 사이트

★ 말줌임표 삼총사 ★

영역 안의 글자가 넘칠 때 자동으로 말줄임표가 나타남.

  • white-space: nowrap; 자동 줄바꿈 금지
  • overflow: hidden; 영역 밖의 내용 숨김
  • text-overflow: ellipsis; 말줌임표(...) 표시

글자 간격 속성들

  • letter-spacing:n; 문자 간격(자간) 지정. 음수 가능
  • word-spacing:n; 단어 간격(띄어쓰기) 지정
