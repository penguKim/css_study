border-radius:n; 영역의 모서리 라운딩

  • border-radius:n; 양사방 같은 값
  • border-radius:n n; 좌측상단과 우측하단 우측상단과 좌측하단이 서로 같은 값
  • border-radius:n n n n; 좌층상단을 시작으로 시계방향으로 각각 다른 값

display 요소가 화면에 나타나는 방식

  1. inline 요소의 특징
    한 줄에 나란히 옴. 크기 지정 불가. 상하여백 지정 불가.
  2. block 요소의 특징
    한 줄에 나란히 올 수 없음. 크기 여백 모두 지정 가능

display의 속성값

  • display:block; block의 성격으로 지정
  • display:inline; inline의 성격으로 지정
  • display:inline-block; inline과 block을 합친 성격으로 지정
  • display:none; 요소가 사라짐
