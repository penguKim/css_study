margin - 요소의 바깥 여백

1. margin의 방향별 지정법

  • margin-top:n; 위쪽만 적용
  • margin-right:n; 우측만 적용
  • margin-bottom:n; 아래쪽만 적용
  • margin-left:n; 좌측만 적용

2. margin의 값 갯수별 지정법

  • margin-:n; 양사방 동일하게 적용
  • margin-:n n; 상하와 좌우가 서로 동일하게 적용
  • margin-:n n n; 상, 좌우, 하 순으로 적용
  • margin-:n n n n; 상,우,하,좌 순으로 적용

1. margin의 방향별 지정법

div01(margin-top: 50px;)
div02(margin-right: 50px;)
div03(margin-bottom: 50px;)
div04(margin-left: 50px;)

2. margin의 값 갯수별 지정법

div01(margin:2em;)
div02(margin:2em 4em;)
div03(margin:2em 4em 1em;)
div04(margin:3em 20% 7em 10%;)
