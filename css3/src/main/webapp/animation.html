animation

@keyframes으로 animation정의하기

@keyframes 애니명 { 0% {속성:속성값;} ... 100% {속성:속성값;} }

animation의 하위속성들

  • animation-name: ; (필수)애니메이션 이름
  • animation-duration:ns/nms; (필수)애니메이션 진행시간
  • animation-delay:ns/nms; 지연 시간
  • animation-iteration-count:n/infinite(무한반복); 애니메이션 반복횟수
  • animation-direction: ; 애니메이션 진행방향
    • normal 정방향(0%~100%)
    • reverse 역방향(100%~0%)
    • alternate 왕복(2회 이상 반복시 0%~100%~0%)
    • alternate-reverse(2회 이상 반복시 100%~0%~100%)
  • animation-timing-function: ; 가속 감속 등
    • ease 기본값. 천천히 시작하고 느리게 끝남.
    • ease-in 시작을 느리게
    • ease-out 끝을 느리게
    • ease-in-out 시작과 끝을 느리게
    • linear 등속
    • cubic-bezier(n,n,n,n) 베지에곡선값을 이용 cubic-bezier.com 생성기 사이트
  • animation-fill-mode: ; 애니메이션 멈춤시 보일 장면
    • none 기본값.
    • forwards 마지막 키프레임 설정값
    • backwards 첫 번째 키프레임 설정값
    • both 양방향
