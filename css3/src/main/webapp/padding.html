padding - 요소의 내부 여백

1. padding의 방향별 지정법

  • padding-top:n; 위쪽만 적용
  • padding-right:n; 우측만 적용
  • padding-bottom:n; 아래쪽만 적용
  • padding-left:n; 좌측만 적용

2. padding의 값 갯수별 지정법

  • padding-:n; 양사방 동일하게 적용
  • padding-:n n; 상하와 좌우가 서로 동일하게 적용
  • padding-:n n n; 상, 좌우, 하 순으로 적용
  • padding-:n n n n; 상,우,하,좌 순으로 적용

1. padding의 방향별 지정법

padding-top: 30px;
padding-right: 20%;
padding-bottom: 2em;
padding-left: 10%

2. padding의 값 갯수별 지정법

padding:50px;
padding: 50px 20%;
padding: 50px 30% 10px;
padding: 30px 30% 1em 10%;
