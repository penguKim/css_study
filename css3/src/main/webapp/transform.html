transform - 변형

변형함수

  • transform: translate(); 위치 변형
  • transform: scale(); 크기 변형
  • transform: rotate(); 회전 변형
  • transform: skew(); 기울기 변형

transform: translate(Xn,Yn); 위치 변형

Xn,Yn은 양수일 때 우측,아래 음수일 때 좌측,위로 이동

transform: translateX(1200px);
transform: translateY(100px);
transform: translateX(-500px);
transform: translateY(-200px);
transform: translate(1000px, -200px);
transform: translate(100%,100%);
[center] 1. position: absolute;
(상위요소는 position:relative;)
left: 50%; top: 50%;
--- 상위요소의 정중앙에서 시작
2. transform: translate(-50%, -50%);
--- 해당요소의 절반만큼 앞, 위로 이동

2. transform: scale(Xn,Yn); 크기 변형

Xn, Yn은 1을 기준으로 1보다 작으면 축소, 크면 확대

transform: scaleX(0.5);
transform: scaleX(2);
transform: scaleY(0.3);
transform: scaleY(2);
transform: scale(0.2,0.2);
transform:scale(2,2);

3. transform:rotate(ndeg); 회전 변형

rotateX, rotateY는 3차원 변형.
각도 단위 deg를 쓰고 음수도 가능.

transform:rotateX(80deg);
transform:rotateY(90deg);
transform:rotateX(180deg);
transform:rotateY(180deg);
transform:rotate(-540deg)
transform: rotate();

4. transform:skew(ndeg); 기울기 변형

각도 단위 deg를 쓰고 음수도 가능.

transform: skewX(30deg);
transform: skewX(-30deg);
transform: skewY(30deg);
transform: skewY(-30deg);
transform: skew(35deg,35deg);
transform: skew();
