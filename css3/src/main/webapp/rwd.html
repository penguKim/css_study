rwd(반응형 웹 디자인)

CSS와 HTML을 사용하여 콘텐츠의 크기를 조정하거나 숨기거나 축소하거나 확대하거나 이동하여 어떤 화면에서도 보기 좋게 만드는 것을 반응형 웹 디자인

viewport 설정

뷰포트는 웹 페이지에서 사용자가 볼 수 있는 영역 <meta name="viewport" content="width=device-width, initial-scale=1.0">
(장치의 화면 너비를 따르도록 페이지 너비를 설정
브라우저가 페이지를 처음 로드할 때 초기 확대/축소 수준을 설정)

Media Queries(미디어 쿼리)

@media 장치유형 and (조건) {
    선택자 {속성:속성값;}
}

장치 유형

  • all - 장치 상관 없음
  • print - 프린터
  • screen -computer screens, tablets, smart-phones
  • speech - 음성낭독기

조건

  • width and height of the viewport - 뷰포트의 너비와 높이
  • width and height of the device - 장치의 너비와 높이
  • orientation (landscape or portrait) - 방향(가로 모드, 세로 모드)
  • resolution - 해상도
